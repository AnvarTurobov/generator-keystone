# generator-keystone 

A [KeystoneJS](http://keystonejs.com) project generator for [Yeoman](http://yeoman.io).

## Getting Started

Create an empty directory for your new KeystoneJS Project, then run the following from it:

```
$ npm install -g yo
$ npm install -g generator-keystone
$ yo keystone
```
